# POIESIS 한정판 운영 가이드 (에디션당 33권)

"에디션당 33개 한정"은 좋은 전략입니다. 희소성은 구매를 앞당기고, 넘버링은 소장 가치를 만들며, 소량 생산은 재고 리스크를 줄입니다.
다만 한정판은 **재고 정확성·신뢰·운영 디테일**이 받쳐줘야 역효과가 안 납니다. 아래는 실제 판매를 위한 추가 절차입니다.

> 사이트에는 이미 반영됨: 에디션당 33권 재고 표시, 남은 수량/판매 progress, 품절 처리, 1인 2권 구매 제한,
> 주문 시 고유 에디션 넘버(No.○○/33) 자동 부여, 품절 시 대기 신청 버튼. (현재는 브라우저 데모 — 아래 1번 참고)

---

## 1. 재고 정확성 — 가장 중요 🔴
한정판의 최대 리스크는 **오버셀(없는 재고 판매)**입니다. 33권뿐인데 35건이 결제되면 신뢰가 무너집니다.

- 현재 데모: 재고가 브라우저(localStorage)에 저장되어 **사용자마다 따로** 계산됩니다 → 실제 판매엔 부적합.
- 실제 운영: **서버 DB에서 재고를 단일 관리**하고, 결제 시점에 재고 차감을 잠금(transaction/lock)해야 동시 주문에도 안전합니다.
- 권장: 결제 성공 웹훅에서 재고 확정 차감, 결제 실패/취소 시 자동 복구.
- 장바구니 담기 ≠ 재고 확보. "장바구니에 담아도 결제 전까지 보장되지 않음"을 안내하거나, 일정 시간 임시 홀드(예: 10분)를 두는 방식 중 선택.

## 2. 에디션 넘버링 & 정품 보증 🟡
- 각 노트에 **No.○○ / 33** 시리얼을 물리적으로 부여 (각인/스탬프/넘버링 스티커/손번호).
- **정품 보증 카드** 동봉: 에디션명, 시리얼, 제작일, 서명/씰. (이미 주문완료 화면·메일에 넘버 표기)
- 진위 확인용 **QR 또는 일련번호 조회 페이지**(추후) — 리셀·선물 시 신뢰도 상승.
- 넘버 배정 규칙 정하기: 결제 순서대로 자동(현재 방식) / 특정 번호(No.1, No.33) 별도 보관·증정 여부.

## 3. 가격·재입고 정책 🟡
- 한정판은 보통 **재입고 없음**이 원칙(희소성 유지). 대신 "다음 에디션"으로 이어가기.
- 정책을 명확히: "이 33권은 재생산하지 않습니다" 또는 "동일 디자인 재입고 가능" 중 택1 → 사이트·FAQ에 명시.
- 33권 × 3 에디션 = 99권 초도 물량. 원가·마진·손익분기 수량 계산 필요(가죽·펜·박스·각인·배송·수수료·반품률 포함).

## 4. 드롭(Drop) 방식 런칭 🟢
한정판은 상시 판매보다 **정해진 시간에 오픈하는 드롭**이 화제성에 유리.
- 사전 알림 신청(이메일/인스타) → 오픈 시각 공지 → 동시 오픈.
- 사이트의 "다음 에디션 알림 신청" 버튼을 실제 이메일 수집과 연결(현재 데모).
- 카운트다운은 **진짜일 때만** 사용. 가짜 타이머/가짜 "n명이 보는 중"은 신뢰를 깎고 표시광고 이슈가 될 수 있음 — 권장하지 않음.

## 5. 품질 관리(QC) 체크리스트 🟡
소량 핸드메이드일수록 검수 기준을 글로 고정하세요.
- 가죽: 치명적 흠집/오염/색 편차 허용 범위 정의 (천연 가죽 특성과 불량의 구분 기준)
- 각인: 위치·농도·번짐 기준, 오각인 폐기 규정
- 제본: 라이플랫 펼침, 풀림/들뜸 확인, 페이지 수·인쇄 품질
- 구성품: 노트·펜(잉크 나옴 확인)·밴드·보증서·박스 누락 점검
- 합격품에만 시리얼 부여 → 재고 등록

## 6. 패키징 & 출고 🟡
- 기프트 박스 손상 방지 완충 포장, 가죽 보호(더스트백/습기 방지재).
- 시리얼-주문 매칭 기록(어떤 번호가 누구에게 갔는지) → CS·진위확인 대비.
- 택배사 계약 + 송장 자동화, 출고 사진 보관(분쟁 대비).

## 7. CS & 정산 🟢
- 자주 묻는 응대 템플릿: 재입고 문의, 시리얼 의미, 가죽 편차, 교환·반품.
- 한정판 반품 시 재판매 가능 여부·시리얼 회수 정책.
- 정산·부가세, 반품 충당금, 에디션별 손익 리포트.

## 8. 마케팅 — "적당히, 정직하게" 🟢
- 핵심 메시지는 이미 강력: **"단 33권, 넘버가 새겨진 단 하나의 노트."** 과장 없이 이 사실만으로 충분.
- 콘텐츠: 제작 과정/각인 영상, 언박싱, 시리얼 클로즈업, 대가 스토리(다빈치·도스토옙스키)와 연결.
- 채널: 인스타그램(비주얼) + 네이버(검색·스마트스토어 병행 검토) + 이메일(드롭 알림).
- 사회적 증거: 실제 구매 후기·평점 수집(현재 후기 영역은 예시 → 실제 리뷰로 교체).
- 선물 수요 공략: "넘버가 있는 선물" 포지셔닝, 메시지 카드 옵션.
